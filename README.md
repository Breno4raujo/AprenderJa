# AprenderJá 📚

> Edtech para Requalificação Profissional de Adultos

## Como rodar

```bash
bun install
bun run dev
```

## Variáveis de ambiente

Crie um arquivo `.env` na raiz:

```env
DATABASE_URL="postgresql://usuario:senha@localhost:5432/aprenderja"
JWT_SECRET="seu-segredo-aqui"
```
